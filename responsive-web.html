Responsive Business Website

The goal was to demonstrate CSS skills using Media Query for responsive design with mobile UI patterns.

  • HTML
  • CSS
  • JS

Live Demo

OVERVIEW

The choice of the topic aimed to highlight the development of a business website that not only prioritizes responsive design but also emphasizes the implementation of dynamic Jquery functions and JavaScript plug-ins.

SITEMAP AND WIREFRAME

Following the research phase, initial design and planning steps were taken. The sitemap was created on Figjam with consideration of the main audience and user group. Afterwards, a low-fi wireframe for each page was created on Figma.

DEVELOPMENT
01

The process was started with a mobile-first approach for an efficient development process. As I started from mobile view first, was able to identify potential challenges early, streamline workflows, and reduce the need for extensive revisions during later stages of development.

02

With the rough structure in place, a Flickity carousel plugin was implemented for dynamic design. In addition, an accordion-style navigation bar was incorporated using JS in order to demonstrate the use of mobile UI design patterns.
